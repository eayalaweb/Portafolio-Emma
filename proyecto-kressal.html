El proyecto consiste en el desarrollo de un sitio web para una escuela de surf que, además de ofrecer información sobre sus cursos y actividades, incluye una tienda online para la venta de productos relacionados con el surf.

➔ WordPress: como CMS principal para la gestión de contenidos.
➔ Constructor Divi: para el diseño visual y la creación de páginas de forma flexible y personalizada.
➔CSS personalizado: para ajustar estilos, mejorar la identidad visual y optimizar la experiencia de usuario.

Objetivo principal: Ofrecer una experiencia inmersiva y creativa, donde el cliente pueda sentirse parte del proceso de fabricación, reforzando así el vínculo emocional con el producto y la marca.
➔Scripts interactivos con JavaScript. para que los cambios de color y diseño se muestren sin recargar la página.
