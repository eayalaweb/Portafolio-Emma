Giroa Restaurante

Este sitio web fue desarrollado como parte de un proyecto escolar, con el objetivo de presentar una experiencia digital elegante para un restaurante ficticio llamado Giroa. El diseño se centra en la alta gastronomía, con un enfoque visual atractivo, imágenes de alta calidad y una estructura limpia.

Se trabajó exclusivamente con HTML y CSS, aplicando principios de diseño responsivo, tipografía sofisticada y organización visual mediante flexbox y grid. El sitio incluye secciones como presentación, carta, bodega, reservas y galería de platos.
